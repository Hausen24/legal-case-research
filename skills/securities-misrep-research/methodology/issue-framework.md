# 方法论：证券虚假陈述裁判问题体系（侵权构成要件五维）

基于 2022 年《最高人民法院关于审理证券市场虚假陈述侵权民事赔偿案件的若干规定》（法释〔2022〕2号，已取消行政处罚/刑事裁判前置程序）。

**本体系是报告第二章的脊柱。** 报告"二、裁判规则全景"按侵权责任构成要件的**五个维度**逐节展开——**（一）责任主体 →（二）虚假陈述行为 →（三）损害结果 →（四）因果关系 →（五）主观过错**，并以"（六）程序事项"作补充。下列 14 个具体问题分布于这五维之下。

**每一维节内固定四段展开**（深度论述内嵌于此，不另设深度剖析章）：
> ① **争点界定**：本维有哪些子问题、争什么、为何重要（含责任形态/认定标准之争）。
> ② **主流裁判规则（含代表案例）**：规则是什么 → 法院凭什么这么判（请求权基础与司法解释条款、法理与举证结构）→ 代表案如何落地。每子问题配 1–2 代表案（脚注带案号链接）。
> ③ **例外与反向情形**：反向/少数观点、抗辩成立个案、推定的推翻路径（配案例）。
> ④ **评析要点**：综合本维、提炼结构性洞察与控辩启示。

> **地域（京沪）差异不是组织轴线**：默认按整体口径呈现规则；是否指出地域分歧由 `stats_guard.divergence_gate` 依样本量裁定，仅在确有不同取向时于相关问题内一句话点出，**不单设地域分歧章节或逐问题小节**。

下列 14 个问题的"问题名"即编码 schema 的键名——编码时 `问题观点` 的键须与本表**一字不差**、顺序按本表（脚本 `ISSUES` 与之一致）。每个问题给出：定义、争点、待提取的"倾向标签"取值集合（供 tidy 编码表与图表统计）。编码时另就该问题填 `争议焦点`/`各方抗辩`（取自 MCP 的 `ControversialFocus`/`DefenseViewpoint`）/`裁判观点`，供 Excel 长表与报告四段写作使用。

## 维一　责任主体（报告"（一）责任主体"）

### 1. 发行人责任
- 定义：发行人（上市公司）的责任性质。
- 争点：过错推定/近乎严格责任、能否免责。
- 倾向标签：承担（过错推定难免责）/ 部分免责 / 免责。

### 2. 控股股东与董监高责任
- 定义：控股股东/实控人、董监高的责任（"追首恶"）。
- 争点：连带责任范围、独董责任、责任比例。
- 倾向标签：连带担责 / 按过错比例 / 免责（注明主体）。

### 3. 中介机构责任
- 定义：保荐承销、会计师、律师、评估等的责任。
- 争点：注意义务边界（"看门人"）、专业事项特别注意/非专业事项一般注意、专业意见可信赖范围、责任比例。
- 倾向标签：承担（注明机构）/ 比例责任（注明比例）/ 免责。

### 4. 责任形态
- 定义：多被告间责任承担方式。
- 争点：连带 / 按份 / 补充 / **比例连带**（新规亮点）。
- 倾向标签：连带 / 按份 / 补充 / 比例连带（注明比例）。

## 维二　虚假陈述行为（报告"（二）虚假陈述行为"）

### 5. 虚假陈述认定
- 定义：是否存在虚假记载／误导性陈述／重大遗漏／不正当披露。
- 争点：信息披露义务范围、披露时点、是否构成"陈述"。
- 倾向标签：构成虚假记载 / 构成误导性陈述 / 构成重大遗漏 / 构成不正当披露 / 不构成。

### 6. 重大性
- 定义：前置程序取消后重大性如何独立认定。
- 争点：价格敏感性标准 vs 投资决策影响标准；价量异动反证出罪。
- 倾向标签：具有重大性 / 不具有重大性；采价格敏感性 / 采投资决策影响 / 二者结合。

### 7. 预测性信息安全港
- 定义：盈利预测、业绩快报等预测性信息是否落入安全港免责。
- 争点：是否充分风险提示、是否善意、是否有合理基础。
- 倾向标签：适用安全港免责 / 不适用（提示不充分/无合理基础/非善意）。

### 8. 自愿性披露
- 定义：非强制披露信息虚假是否担责、标准是否不同。
- 争点：自愿披露的真实准确完整义务、对市场的实际影响。
- 倾向标签：自愿披露同样担责 / 标准从宽 / 不担责。

## 维三　损害结果（报告"（三）损害结果（损失）"）

### 9. 损失计算
- 定义：投资差额损失算法、基准价与基准日、佣金印花税利息。
- 争点：移动加权平均 vs 先进先出；基准价口径；损失范围。
- 倾向标签：移动加权平均法 / 实际成本法 / 其他；含/不含佣金印花税利息。

## 维四　因果关系（报告"（四）因果关系"）

### 10. 揭露日认定
- 定义：实施日、揭露日/更正日、基准日的确定（重点为揭露日，是交易/损失因果与损失计算的时点前提）。
- 争点：立案调查（告知）日 / 媒体报道日 / 自我更正日 / 行政处罚决定日，何者为揭露日（首次性＋警示性取最早）。
- 倾向标签：立案调查公告日 / 媒体首次报道日 / 自我更正日 / 处罚决定日 / 其他。

### 11. 交易因果关系
- 定义：交易与虚假陈述之因果（推定成立 + 被告反证）。
- 争点：反证事由（投资者专业性/明知、价格已反映、诱多诱空型区分等）。
- 倾向标签：推定成立未被推翻 / 反证成立（注明事由）/ 诱空型特殊处理。

### 12. 损失因果关系
- 定义：损失中属虚假陈述所致部分的认定（系统风险/他因剔除）。
- 争点：系统风险扣除、行业/板块风险、自身经营风险等他因剔除及比例、举证责任。
- 倾向标签：系统风险扣除比例（记 % 入 `系统风险扣除比例`）/ 扣除非系统他因 / 不予扣除 / 全额。

## 维五　主观过错（报告"（五）主观过错"）

### 13. 主观过错与勤勉尽责抗辩
- 定义：各责任主体的过错认定与免责抗辩（与维一互补：维一看是否担责/责任形态，本维看过错如何认定、能否以勤勉尽责免责）。
- 争点：过错推定与举证责任倒置、勤勉尽责/善意合理信赖标准、看门人注意义务的过错门槛、独董抗辩。
- 倾向标签：过错推定成立 / 勤勉尽责免责（注明主体）/ 善意合理信赖免责 / 重大过错 / 一般过错。

## 程序事项（报告"（六）程序事项（补充）"，非实体构成要件）

### 14. 诉讼时效与示范判决机制
- 定义：诉讼时效起算、示范判决/代表人诉讼（普通/特别）程序适用。
- 争点：时效起算点、示范判决既判力对平行案件的影响、代表人范围。
- 倾向标签：时效起算（揭露日/知道应知日）；示范判决 / 普通代表人诉讼 / 特别代表人诉讼 / 普通共同诉讼。

## 编码与解读提示
- 一案通常只涉及其中数个问题，未涉及的略去，不强行填。
- "倾向标签"是给统计用的离散值，尽量归入上述集合；确需新值时保持口径一致，写入 tidy 编码表。
- `系统风险扣除比例` 单独存数值（0–1），便于整体分布。
- **样本量自适应**：`stats_guard.depth_mode` 为 `qualitative_deep`（案少）时，重心在逐问题深挖法院论证；为 `quantitative_lead`（案多）时，量化分析承载全景、定性深挖只取标杆/分歧案。
- 报告对"分歧/反向"列与控辩要点须与正文一致；无实质分歧时如实写"本批无明显分歧"，不得为凑差异而编造。
