# report-plaintiff.md — 攻方场景（权利人 / 原告视角）· 生成模板（v2 策略增强版）

> 目标：站在权利人立场回答"怎么赢"。基于全样本客观数据，但分析重心落到——主流规则中对我方有利之处、应主张的论点与证据、对方抗辩的破解、可援引的有利判例、风险点。
> **本模板复用居中模板 `report-neutral.md` 的 v2 深度内核**（§0 全局铁律、§2 每焦点四段深度、§3 脚注引注、§5 图表、§6 设计系统、§7 封面元信息、密度受 `depth_mode` 调控），下面只写攻方的**差异点**。生成前务必先读 `report-neutral.md`。

## 攻方差异点

### 一、研究范围与方法
同居中模板（简版；详细方法论入附录二）。

### 二、整体形势研判（落到"这对原告意味着什么"）
把整体概况与交叉关联、抗辩有效性、判赔、年度演进的发现，**全部解读成对权利人的含义**（化成裁判规律句，不出现"维度X"字样）：
- 整体支持率：原告在该类案件中的总体赢面（配 `![chart:overview]`）。
- 对方常用抗辩的成功率：成功率低 = 可放心预判其失败；成功率高 = 须重点防范。
- 影响判赔高低的因素 → 我方如何主张以争取高判赔。
- 当前裁判趋势是否对权利人越来越有利（配 `![chart:result_year]`）。
- 推断性措辞受样本量闸门约束（小样本标"示取向、非占比定论"）。

### 三、各争议焦点的攻方策略（每焦点四段 + 攻方落点）
每个焦点仍按居中 §2 的四段（争点界定 / 主流规则含代表案 / 例外反向含代表案 / 小结）写足深度；在小结后加**攻方落点**三小条：
1. **我方应主张的论点与证据**：要赢需证明什么（结合胜诉案中法院认定的关键事实）。
2. **对方抗辩及破解**：列该焦点下对方常见抗辩，逐一给出已被法院否定的理由（脚注引否定该抗辩的判例）。
3. **风险点**：少数支持被告的案例提示了什么风险、如何规避。

### 五、结论：整体赢面与最优攻击路径
- 整体赢面与最优攻击路径；哪几个论点是制胜关键（要素-结果影响度的法律解读）；判赔最大化的主张要点。
