# 最高人民法院指导性案例本地数据集

**用途**：实案研究模式顺位①（法发〔2020〕24号第四条）的全文来源——北大法宝 MCP 对指导性
案例不返回正文，本数据集补齐"深度比对"所需的裁判要点/基本案情/裁判理由全文。

- **来源**：最高人民法院官网"审判业务→指导性案例"栏目
  （https://www.court.gov.cn/shenpan/gengduo/77.html ），由
  `scripts/fetch_guiding_cases.py` 限速合规抓取（指导性案例属官方发布的司法文书性质材料，
  不适用《著作权法》保护，见该法第五条）。
- **结构**：`index.json`（编号/标题/链接/发布时间索引）+ `cases/<编号>.json`
  （关键词、裁判要点、基本案情、裁判结果、裁判理由、相关法条、全文）。
- **更新**：最高法分批发布、更新缓慢。定期重跑 `python3 scripts/fetch_guiding_cases.py`
  即增量补抓新批次。

## 缺失编号说明（重要）

官网栏目当前未收录以下编号（抓取时点）：10、11、12、15、17、18、19、20、21、22、23、
30、45、145、231、236。其中多数与最高人民法院历次**废止公告**的批次特征吻合
（如因《民法典》施行被废止的早期民商事指导性案例），疑似废止后从栏目下架；个别可能系
发布渠道差异。**本数据集不维护效力状态**——按《类案检索规定》第九条，引用任何指导性
案例前均须核查其是否与新法律/司法解释冲突、是否被新指导性案例取代或已被废止
（可经法宝/官网公告核对），SKILL 已将该核查设为引用前置步骤。
