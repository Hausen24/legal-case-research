# 更新日志

本项目版本号遵循「功能里程碑」粒度，非严格 SemVer。

## v0.4 —— 开源加固（可展示 / 可复现 / 可证明）
- **新增 LICENSE（MIT）**，使项目可被合法使用与二次开发。
- **新增脱敏演示样本** `examples/`：合成一份与 MCP 返回结构一致的小样本（虚构案号/当事人，见 `DISCLAIMER.md`），**无需法宝 Token 即可跑通整条管道**并产出报告/Excel/图表，作为 GitHub 展示与测试夹具。
- **新增反幻觉收尾自检** `scripts/verify_report.py`：扫描成品报告每一个案号，逐一比对样本池，引用了样本池外的案号即报错——把"承诺不编案号"升级为"机器证明可溯源"。已接入两套技能的收尾流程与 CI。
- **新增自动化测试与 CI**：`tests/` 下 pytest 覆盖公共派生、样本量自适应闸门（含"小样本不作分歧推断"回归保护）、Excel 实质列非空、反幻觉校验正反用例；GitHub Actions 每次推送/PR 运行测试 + 演示管道冒烟 + 反幻觉校验。
- README 增补徽章、英文简介、Demo 展示区（图表内联）、可靠性章节。
