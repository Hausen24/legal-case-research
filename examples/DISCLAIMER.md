# 演示数据免责声明

本目录（`examples/`）下的全部数据**均为合成的演示数据**，用于在没有北大法宝订阅 Token 的情况下，
向访客展示本工作流的产出形态与脚本链路。

- **案号**：统一使用并不存在的「示」字头法院代字（如 `（2023）示01民初0001号`），不对应任何真实案号。
- **当事人**：均为「甲影视公司 / 乙科技公司」等占位名称，不指向任何真实主体。
- **裁判事实与说理**：为演示目的编写，**不代表任何真实判决**，不得用于任何法律目的或引用。
- **链接**：指向 `demo.example.com` 占位域名，并非真实法宝链接。

真实研究请在仓库根目录配置 `.mcp.json`（见 `.mcp.json.example`），由工作流从北大法宝 MCP 实时检索；
真实产出默认落在被 `.gitignore` 的 `research/` 目录，不会进入版本库。

> 本工作流的核心纪律之一是**反幻觉**（引用必带可溯源案号+法院+链接）。此演示数据虽为虚构，
> 但其字段结构与真实 MCP 返回完全一致，`scripts/verify_report.py` 对演示报告的案号校验同样成立——
> 即"报告引用的每个案号都能在样本池中溯源"。
